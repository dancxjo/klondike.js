This is an implementation of Klondike using AngularJS.

The design is like this:

Game Service
Stores the model for the game.
Game Controller
Attach to the Game Service. Shuffles the deck. Attaches to the main view. Lays out the tableau.
Card Directive
Handles drawing a card. Handles dragging talking to Game Service.
Receptacle Directive
Handles receiving a drop talking to the Game Service.
Various Filters
Various filters related to drawing a card.
